## Scalar data types

### integer data type

int: unlimited precision signed integer

```markdown
>>> 10
10
>>> type(10)
<class 'int'>
>>> x = 15
>>>x
15
>>> type(x)
<class 'int'>
>>> 0b1010
10
>>> bin(10)
'0b1010'
>>> 0o10
8
>>> oct(8)
'0o10'
>>> 0x145
325
>>> hex(325)
'0x145'
>>> int(3.5)
3
>>> int(-3.5)
-3
>>> int("496")
496
>>> int("1000",3)
81
```
